## Tips for Managing the Bot

### Setting Bot Permissions
- Create a dedicated channel like `#kvk-stats` for bot usage
- Set permissions so the bot can:
  - Read Messages
  - Send Messages
  - Embed Links
  - Use Slash Commands

### Encouraging Usage
- Pin the announcement in relevant channels
- Share your own stats first to lead by example
- Recognize top performers weekly
- Use `/top` command during KvK events

### Monitoring Usage
- Check Railway logs: https://railway.app/dashboard
- Look for common errors in bot responses
- Ask members for feedback on features they'd like
